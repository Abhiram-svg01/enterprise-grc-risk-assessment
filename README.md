# Enterprise Cyber Risk Assessment & Compliance Project

This repository demonstrates an enterprise-level cyber risk assessment conducted for a simulated mid-sized healthcare organization.

The project follows **NIST SP 800-30** methodology for risk assessment, with control alignment to **NIST SP 800-53** and **ISO/IEC 27001**.

---

## Scope
- Electronic Health Record (EHR) system
- Patient database
- Cloud storage environment
- Internal networks and endpoints
- Third-party billing vendor

---

## Key Activities
- Asset identification and risk assessment
- Threat and vulnerability analysis
- Risk scoring based on likelihood and impact
- Control mapping to NIST SP 800-53 and ISO 27001
- Gap analysis and remediation planning
- Executive-level risk reporting

---

## Deliverables
- Risk Register
- Control Mapping Matrix
- Gap Analysis & Remediation Plan
- Executive Cyber Risk Assessment Summary

> **Note:** This project is a simulated scenario created for professional portfolio demonstration purposes only.
The Excel workbook consolidates the asset inventory, risk register, control mapping, and remediation plan into a single enterprise-style GRC assessment.
